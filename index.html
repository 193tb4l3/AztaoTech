Panduan Penggunaan
Fitur Utama:
  • Analisis statistik lengkap (mean, median, modus, standar deviasi)
  • Tabel distribusi frekuensi dan kumulatif
  • Visualisasi data interaktif (histogram, poligon, ogive, pie chart)
  • Kalkulator Z-Score
  • Import data dari file Excel/CSV
  • Ekspor hasil analisis
  • Simpan hasil untuk dibuka kembali
  • Dukungan untuk data tunggal dan data berkelompok
Cara Penggunaan:
  1. Pilih jenis data (tunggal atau berkelompok)
  2. Masukkan data:
    • Untuk data tunggal: masukkan nilai numerik
    • Untuk data berkelompok: masukkan interval kelas dan frekuensi
  3. Klik tombol "Analisis Data" untuk memproses
  4. Gunakan "Contoh Data" untuk mencoba fitur
  5. Navigasi antar tab untuk melihat berbagai visualisasi
  6. Gunakan kalkulator Z-Score untuk analisis lebih lanjut
  7. Ekspor atau simpan hasil analisis
Tip: Anda bisa mengubah tema gelap/terang dengan tombol di pojok kanan atas.
